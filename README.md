# Top 100 Albums

## Description
This web app displays top 100 albums from iTunes/Apple Music. 
[Click here visit the Heroku page!](https://itunes-top100-sp.herokuapp.com/)

## Built with
- React
- Node.js
- SCSS

## Other Frameworks/Features
- Ant Dsign
- Foundation
- CSS animations
- CSS media queries
